# Rota Viagem API

API REST em .NET 8 para consultar a melhor rota de viagem entre cidades, com persistência em memória, Swagger e testes com xUnit.

## Funcionalidades

- CRUD de rotas
- Consulta da melhor rota com menor custo
- Swagger UI
- Testes unitários

## Executando

```bash
dotnet run --project RotaViagem
```
